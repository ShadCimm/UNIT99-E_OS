# Configurable Settings

All settings are found under `SET`.

## DISPLAY

Configurable:

- Theme color: `GREEN`, `AMBER`, or `WHITE`.
- Auto theme based on time of day.
- Day color.
- Night color.
- Day start hour.
- Night start hour.
- Text size, from 70% to 150%.
- UI scale, from 70% to 150%.
- Vault Boy Pip slideshow interval, from 0.5s to 10s.
- Vault Boy Pip image scale, from 100% to 150%.

## MAP

Configurable:

- Primary map region.
- Active map regions.
- Favorite map regions.
- Downloaded map packages.
- Manual file-loaded map packages.
- Package deletion.

## AUDIO

Configurable:

- FX volume, used for Pip-OS UI sounds and related effects.

## HUD

Configurable:

- GPS units: `MPH` or `KPH`.
- Show/hide top speed.
- Show/hide GPS indicator.
- Show/hide Bluetooth indicator.
- Show/hide CPU usage.
- Show/hide memory usage.

## EFFECTS

Configurable:

- Scan lines.
- CRT screen bleed.
- Low power mode.
- Live radio waveform.
- Waveform refresh rate, from 5 FPS to 60 FPS.
- Random section phosphor burst.
- Random phosphor probability, from 0% to 100%.
- Vault Boy watermark.
- Vault-Tec logo watermark.
- Vault-Tec name watermark.
- Middle watermark size, from 50% to 300%.
- Watermark opacity, from 0% to 200%.

## APPS

Configurable:

- Notification access shortcut for external media sessions.
- Four media shortcut apps for dashboard media widgets.
- Speed touch action.
- Four speedometer shortcut apps.
- Four top-bar shortcut apps under the Vault-Tec logo menu.

Speed touch actions:

- `SHOW RECENTS MENU`
- `JUMP TO LAST RECENT APP`
- `OPEN SYSTEM RECENTS`
- `OPEN ANDROID AUTO`
- `OPEN CARPLAY`

## OBD-II

Configurable:

- Preferred paired Bluetooth OBD-II device.
- Auto-connect on launch.
- Auto-connect retry interval, from 5 seconds to 300 seconds.
- Manual connect/disconnect.
- Read DTCs when connected.

## STATS

Configurable gauge maximums:

- Max speed, 50 to 500 KPH.
- Max RPM, 1000 to 15000.
- Max coolant, 80C to 180C.
- Max voltage, 8V to 30V.
- Max intake temperature, 20C to 200C.
- Max airflow, 50 to 1000 g/s.

Configurable warning behavior:

- Radiation red alert audio.
- Radiation ramp multiplier, 0.1x to 2.0x.
- RPM red warning, 1000 to 15000 RPM.
- Low voltage warning, 8V to 20V.
- High voltage warning, 8V to 30V.
- Coolant hot warning, 80C to 140C.

## DEV

Advanced/testing settings:

- Mock GPS routes:
- `SPAIN ROAD RUN`
- `CALIFORNIA HIGHWAY`
- `LAS VEGAS CITY`
- Demo OBD data.
- Disable boot preloading.

## SYSTEM

Configurable:

- Exit launcher. This closes/removes the Pip-OS task so you can return to another launcher or system flow.

## ABOUT

Shows:

- Project information.
- App version.
- Support/social links.
- Safety and warranty notes.
- Fan-project rights/disclaimer text.
